## Using Image Pipeline 









#### Debugging
 - Viewing an image topic
   - Using RViz:
     -  launch rviz using `rosrun rviz rviz` 
     - Add (bottom left) > By topic (in the pop-up menu) > Select the relevant image
   - Using image_view:
     - launch image view 
        ```
        rosrun image_view image_view image:=/<<topic_name>>
        ```
